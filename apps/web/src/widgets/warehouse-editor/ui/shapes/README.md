# warehouse-editor/ui/shapes

This folder is the warehouse-editor scene rendering layer.

It is not `shared/ui`.

## Scope

- Scene projection/rendering for rack, section, and cell visuals.
- Rendering and overlay composition specific to warehouse-editor canvas behavior.

## Entity Note

Section is not a first-class entity yet.
